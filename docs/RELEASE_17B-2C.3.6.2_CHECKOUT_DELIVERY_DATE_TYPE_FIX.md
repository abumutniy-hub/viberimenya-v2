# RELEASE 17B-2C.3.6.2 — Checkout delivery date type fix

Исправляет остановку релиза 17B-2C.3.6.1 на полном TypeScript typecheck.

Причина: серверная проверка завершившегося интервала обращалась к отсутствующему полю
`body.deliveryDateText`, хотя схема запроса содержит `body.deliveryDate`.

Исправление:
- используется существующее поле `deliveryDate`;
- source-contract отдельно запрещает повторное появление `deliveryDateText` в этом вызове;
- вся логика гостевого checkout, офлайн-оплаты, самовывоза, интервалов, мобильного интерфейса и Telegram pairing сохранена;
- схема базы данных и миграции не меняются.
