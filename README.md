# ВЫБЕРИ МЕНЯ v2

Чистая новая версия проекта: интернет-магазин цветов, CRM, Telegram-бот и SaaS-ready архитектура.

## Главный принцип

Первая версия запускается как один магазин: **ВЫБЕРИ МЕНЯ**.

Внутри проект сразу проектируется как будущая SaaS-платформа для нескольких магазинов через `shop_id`.

## Структура

- `apps/web` — сайт, каталог, корзина, личный кабинет, CRM-интерфейс
- `apps/api` — API, заказы, оплата, доставка, склад, роли
- `apps/bot` — Telegram-бот для клиентов и сотрудников
- `packages/db` — база данных, схема, миграции
- `packages/ui` — общие UI-компоненты
- `packages/config` — общие настройки и константы
- `storage` — загружаемые файлы
- `docs` — документация
- `scripts` — служебные скрипты

## Правила проекта

1. Не добавлять тестовые тексты в production-код.
2. Тестовые данные — только отдельным seed-скриптом.
3. Все основные таблицы должны быть SaaS-ready через `shop_id`.
4. Главная страница создаётся сразу как финальная витрина.
5. Мобильная версия строится как приложение с нижним меню.
